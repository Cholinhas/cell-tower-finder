# 📡 Cell Tower Finder

Buscador de antenas de telefonía móvil cercanas a cualquier ubicación del mundo utilizando la base de datos global de [OpenCellID](https://opencellid.org).

## ✨ Características

- 🔍 Búsqueda interactiva por consola
- 🌍 Coordenadas válidas en cualquier país
- 📡 Muestra información detallada de las antenas (Cell ID, operador, rango)
- 🗺️ Genera un mapa interactivo con tu ubicación y todas las antenas encontradas
- 🚀 Ligero y fácil de usar

## 📋 Requisitos

- Python 3.7 o superior
- Conexión a Internet
